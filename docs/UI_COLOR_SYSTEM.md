# UI Color System

## Design Philosophy
Hotel Management and Reservation System
Style:
* Professional
* Modern
* Premium
* Clean
* Enterprise Dashboard

---

# Primary Colors
* **Primary 500**: `#2563EB` (Main Buttons, Primary Actions, Links, Active Menu)
* **Primary Hover**: `#1D4ED8` (Button Hover, Active States)
* **Primary Light**: `#DBEAFE` (Selected Rows, Light Background)

# Secondary Colors
* **Secondary**: `#0F172A` (Sidebar, Header, Dashboard Navigation)
* **Secondary Light**: `#334155` (Sidebar Hover)

# Accent Colors
* **Gold Accent**: `#D4AF37` (Premium Badge, VIP Room, Hotel Branding, Subscription Status)

# Success Colors
* **Success**: `#16A34A` (Reservation Confirmed, Payment Success, Check-in Success)
* **Success Light**: `#DCFCE7` (Success Alerts)

# Warning Colors
* **Warning**: `#F59E0B` (Pending Reservation, Upcoming Checkout)
* **Warning Light**: `#FEF3C7` (Warning Alerts)

# Error Colors
* **Danger**: `#DC2626` (Cancel Reservation, Delete Actions, Validation Errors)
* **Danger Light**: `#FEE2E2` (Error Notifications)

# Neutral Colors
* **Background**: `#F8FAFC` (Main Background)
* **Card Background**: `#FFFFFF` (Cards, Tables, Forms)
* **Border**: `#E2E8F0` (Inputs, Cards, Tables)

# Typography Colors
* **Heading**: `#0F172A` (Titles, Headings)
* **Body Text**: `#334155` (Normal Text)
* **Muted Text**: `#64748B` (Descriptions, Secondary Information)

# Reservation Status Colors
* **AVAILABLE**: `#16A34A` (Green)
* **RESERVED**: `#F59E0B` (Orange)
* **OCCUPIED**: `#2563EB` (Blue)
* **MAINTENANCE**: `#DC2626` (Red)

# Dashboard Charts
* **Revenue**: `#2563EB`
* **Occupancy**: `#16A34A`
* **Reservation Trend**: `#F59E0B`
* **AI Analytics**: `#8B5CF6`

# Dark Mode (Future)
* **Background**: `#0F172A`
* **Card**: `#1E293B`
* **Text**: `#F8FAFC`
* **Border**: `#334155`

# UI Rules
* Use Primary Blue for actions.
* Use Gold only for premium features.
* Use Green for successful operations.
* Use Red only for destructive actions.
* Maintain WCAG accessibility standards.
* Keep consistent spacing and colors across all screens.
