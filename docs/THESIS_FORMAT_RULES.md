# THESIS_FORMAT_RULES.md

## Thesis Language

Primary language: Vietnamese

Technical terms may remain in English when commonly used:

* Angular
* Spring Boot
* JWT
* REST API
* Docker
* SQL Server

All explanations, descriptions and analysis must be written in Vietnamese.

---

## Page Format

Paper Size:
A4

Print:
Single-sided

Margins:

Left: 3 cm

Right: 2 cm

Top: 2 cm

Bottom: 2 cm

---

## Font

Font Family:

Times New Roman

Body Text:

Size: 13

Line Spacing:

1.5 lines

Paragraph:

Justified

First Line Indent:

1.0 cm

---

## Page Number

Position:

Bottom Center

---

## Chapter Format

Chapter Title:

Font Size: 14

Bold

Uppercase

Centered

Example:

CHƯƠNG 1
TỔNG QUAN ĐỀ TÀI

---

## Level 1 Heading

Font Size: 13

Bold

Uppercase

Left Align

Example:

1.1. MỤC TIÊU NGHIÊN CỨU

---

## Level 2 Heading

Font Size: 13

Bold

Sentence Case

Left Align

Example:

1.1.1. Mục tiêu tổng quát

---

## Figure Rules

Figure Caption Position:

Below Figure

Example:

Hình 3.1. Sơ đồ Use Case tổng quát

Every figure must be referenced in text.

---

## Table Rules

Table Title Position:

Above Table

Example:

Bảng 3.1. Danh sách tác nhân hệ thống

Every table must be referenced in text.

---

## UML Rules

Required UML:

Use Case Diagram

Activity Diagram

Sequence Diagram

Class Diagram

ERD Diagram

Every diagram requires:

Purpose

Description

Analysis

Conclusion

---

## Screenshot Rules

Each implemented screen requires:

Screen image

Screen description

Functions description

User role description

---

## Citation Rules

Only cited references are allowed.

Sort references alphabetically.

Use consistent citation style.

---

## Thesis Length

Minimum:

50 pages

Maximum:

100 pages

Excluding:

Cover

Appendix

---

## AI Documentation Rules

Whenever a feature is completed:

Generate:

1. Thesis section

2. Figure captions

3. Table captions

4. UML explanation

5. Implementation explanation

6. Testing explanation

Use academic Vietnamese writing style.
